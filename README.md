# Software requirements
Postgres 10.1
Gradle 4.4.1
JDK 8 (With path variables set to JDK)

# Build Instructions
```groovy
./gradlew run
```

# Docker build instructions
```
docker-compose up
```

# Local URLs

## Adminer
1) Windows: http://192.168.99.100:8081/
2) Linux: http://localhost:8081/

## Postgres
1) Windows: 192.0.0.1:5432
2) Linux: localhost:5432
